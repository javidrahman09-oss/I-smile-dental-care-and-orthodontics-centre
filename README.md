# I Smile Dental Care and Orthodontic Centre

A modern, responsive website for a dental clinic based in Chennai. This project showcases dental services, professional expertise, and a patient-focused digital experience.

## 🚀 Live Demo
Check out the live website here: [https://i-smile-dental-care-and-orthodontic.vercel.app/](https://i-smile-dental-care-and-orthodontic.vercel.app/)

## ✨ Key Features
* **Service Overview:** Detailed sections for Orthodontics, General Dentistry, and Digital Consultations.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **Booking Integration:** Easy "Book Consultation" calls-to-action for patient lead generation.
* **FAQ Section:** Interactive section to answer common patient concerns.

## 🛠️ Built With
* **HTML5** - Semantic structure
* **CSS3** - Custom styling and layout
* **Vercel** - Automated deployment and hosting
* **GitHub** - Version control
